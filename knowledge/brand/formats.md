# BodyBase — Formatos de Tweet

## Framework Base: Tese → Consequência → Prova → Revelação → CTA

Todo tweet bem-construído da BodyBase segue esta lógica, comprimida ou expandida:

1. **Tese** — afirmação forte, dado ou contrário ao intuitivo
2. **Consequência** — por que isso importa para o leitor
3. **Prova** — fonte, dado, mecanismo
4. **Revelação** — o insight que o leitor não esperava
5. **CTA** — o próximo pensamento, não necessariamente um link

---

## Formato 1 — Dado Surpreendente

**Objetivo**: iniciar com estatística ou fato que cria dissonância cognitiva

**Estrutura:**
```
[Número ou fato contraintuitivo]

[Contexto que explica o dado]

[Mecanismo ou causa]

[Implicação prática ou pergunta que fica no ar]

(Fonte, ano)
```

**Exemplo BOM:**
```
Pessoas com VO2max no quartil superior vivem em média 5 anos a mais
do que as do quartil inferior — independente de IMC.

Não é o peso que prediz longevidade. É a capacidade cardiovascular.

O mecanismo: mitocôndrias eficientes = menor estresse oxidativo ao longo de décadas.

Seu VO2max estimado está no radar dos seus exames?

(JAMA, 2018 — Kokkinos et al., n=122.000)
```

**Exemplo RUIM:**
```
VO2max é super importante para sua saúde! 🏃
Especialistas recomendam que você faça exercício cardio todo dia.
Consulte seu médico para saber mais!
```
*Por quê é ruim: vago, sem dado, sem fonte, tom de conselho genérico*

## Formato 2 — Opinião Contrária

**Objetivo**: desafiar crença comum com base em evidência

**Estrutura:**
```
[Crença popular que o leitor provavelmente tem]

[Dado que contradiz ou nuança]

[Por que a crença popular persiste]

[O que os dados realmente dizem]

[Conclusão calibrada — não absolutista]
```

**Exemplo BOM:**
```
"Colesterol alto é o vilão das doenças cardíacas."

Depende muito de qual colesterol.

LDL-P (partículas de LDL) é mais preditivo de risco
do que LDL-C (o número padrão nos exames).

Você pode ter LDL-C "normal" com LDL-P elevado —
e vice-versa. São coisas diferentes.

A maioria dos exames de rotina no Brasil só mede LDL-C.

(MESA Study, 2005; revisado em JACC 2019)
```

**Exemplo RUIM:**
```
Você sabia que o colesterol não é tão ruim assim?
A mídia distorceu tudo!
Siga nossa página para descobrir a verdade sobre sua saúde!
```
*Por quê é ruim: conspiracionista, sem dado, "a verdade" = red flag*

## Formato 3 — Thread Educativa

**Objetivo**: explorar um tema em profundidade, criar autoridade

**Estrutura do tweet 1 (gancho):**
```
[Promessa clara do que a thread vai entregar]
[Por que vale ler — benefício direto para o leitor]
🧵 [n] tweets
```

**Estrutura dos tweets 2-N (desenvolvimento):**
- Cada tweet = 1 conceito ou dado
- Último item antes do fim = o mais surpreendente
- Formato consistente: conceito → dado → fonte

**Tweet final:**
```
[Síntese em 1-2 linhas]
[Pergunta ou convite à reflexão — não link]
```

**Exemplo de gancho BOM:**
```
HRV (variabilidade de frequência cardíaca) é o biomarcador
mais subestimado da medicina preventiva.

Prediz infarto, recuperação, longevidade — e você pode medir de graça.

O que a literatura diz sobre ele:
🧵 8 tweets
```

**Exemplo de gancho RUIM:**
```
Vamos falar sobre HRV hoje!
É muito importante para a saúde.
Thread 👇
```
*Por quê é ruim: não diz por que ler, sem dado, genérico*

## Formato 4 — Notícia Comentada

**Objetivo**: comentar pesquisa recente com análise própria

**Estrutura:**
```
[O que o estudo encontrou — 1 linha]

[Contexto: por que este estudo importa / o que ele adiciona]

[Limitação principal do estudo]

[O que isso muda na prática (ou não muda)]

(Revista, mês/ano — link ou DOI se couber)
```

**Exemplo BOM:**
```
Novo estudo em Nature Aging: intervenção com rapamicina em cães idosos
mostrou extensão de healthspan sem toxicidade significativa (n=580).

Relevante porque é o primeiro RCT de longa duração com rapamicina em mamíferos grandes.

Limitação: cães ≠ humanos; mecanismos de mTOR diferem.

Não muda recomendações para humanos hoje —
mas reforça a via mTOR como alvo legítimo para longevidade.

(Nature Aging, jan 2024)
```

**Exemplo RUIM:**
```
INCRÍVEL! Cientistas descobriram a chave para viver mais!
Este estudo vai mudar tudo que você sabe sobre envelhecimento! 🔥
```
*Por quê é ruim: sensacionalista, sem dado preciso, clickbait*

## Regras Gerais de Formato

### Comprimento
- Tweet standalone: 180-240 caracteres idealmente (não usar todos os 280)
- Thread: 6-10 tweets; raramente mais
- Cada tweet da thread: autossuficiente (pode ser screenshotado isolado)

### Emojis
- Uso criterioso: máximo 1-2 por tweet
- Aceitáveis: 🧵 (thread), 📊 (dado), → (transição lógica)
- Evitar: 🔥💪✨🙌 (parecem conteúdo de coach)

### CTA (Call to Action)
- Nunca: "Compre agora", "Acesse o link na bio"
- Aceito: pergunta ao leitor ("Você monitora isso?"), convite à reflexão
- Aceito: "Salva esse tweet" quando o conteúdo é genuinamente útil
- Aceito: "Compartilha se fez sentido" — sem forçar
